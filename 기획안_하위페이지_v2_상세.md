|---|---|---|
| 카피 | 형용사 나열·"맛있는 ○○ 주문하세요"·지어낸 문구 | **브랜드 실물 카피(6부작·브로슈어) 재사용**, 절제, 없으면 비워둠 |
| 타이포 | 고딕 한 종·디폴트 자간 | 아리따부리 명조 + Pretendard 위계, 넉넉한 자간·행간 |
| 사진 | 스톡·AI컷·톤 제각각·접시 클로즈업만 | 톤 통일 실사(정식·게장·매장·공정·손), 라이프스타일 |
| 레이아웃 | 균일 카드 그리드 남발·꽉 참 | 여백 큰 위계, 섹션 리듬, 히어로가 먼저 말함 |
| 컬러 | 원색 다수 | 크림/우드 + 인장적색 1포인트(상페 오렌지 미사용) |
| 모션 | 자동 슬라이드·과한 팝업 | 스크롤 반응 절제 페이드(메인 IntersectionObserver 승계) |
| em-dash | — 사용 | 금지(쉼표·중점·줄바꿈) |

**최종 판정 방법**: 착수하면 **공통 템플릿 + 창업안내 1장을 먼저 렌더 → 데스크탑/모바일 캡처를 눈으로 확인**(이게 "완성도/AI티" 판정의 유일한 방법). 통과 후 나머지 4페이지.

## 10. 디자인 요소 운용 — 아이콘·라인·장식 (실행 지침)
> 핵심: **제네릭 아이콘폰트(FontAwesome)·이모지·컬러/입체 아이콘 금지.** 이게 AI/템플릿 티의 1순위. 브랜드가 이미 가진 장식 언어를 끌어쓰고, 기능 아이콘만 단일 스타일로 자작.

### 10A. 장식 모티프 (브랜드 소유 자산 재사용 = 최고의 anti-AI)
| 자산 | 실물 | 하위 적용 |
|---|---|---|
| **낙관 도장** `seal.png` | 전서체 인장 | 섹션/카피 끝 서명, 히어로 하단 |
| **한옥 지붕라인** `deco-roofline-ink.png` | 기와 라인 | 페이지 히어로 크라운, 섹션 상단 구분 장식 |
| **수채화 일러스트** (우리의 이야기 6부작) | 게·참기름병·나물·맞잡은손·보리·상차림 | brand 스토리·menu 재료·먹는법 삽화 (PDF에서 추출 필요) |
| **line-art 워터마크** (상페) | 게·보리·마늘 얇은 선화 | 섹션 배경 옅은 워터마크(옅게 .06~.1) |
| **곡물(보리) 그래픽** | 이삭 다발 | 푸터 위·섹션 구분 오너먼트 |

### 10B. 라인/괘선 시스템 (메인 style.css 승계)
- 소제목 밑 **28px 인장적색 구분선**(`h3::after` 그대로) — 카드·섹션 제목 공통 시그니처
- **세로 괘선(vertical rule)** — 우리의이야기/상페식. 챕터 라벨·인용·섹션 구분에 얇게(1px, ink 20%)
- **헤어라인 보더**(ink 12~13%) — 매장 정보표·메뉴 리스트·절차 구분(메인 menu-list 톤과 동일)
- 챕터 라벨 위 **짧은 rule**(24px) — 섹션 진입 사인 보조

### 10C. 기능 아이콘 (리서치 확정 — Lucide 기반)
- **세트 = Lucide** (ISC 라이선스, 상업 무료·표기 불필요, ~1,600개, 단일 웨이트 아웃라인). 백업=Phosphor "thin". **한 세트만, 절대 혼용 금지.**
- **스타일 규칙(엄격)**: `strokeWidth=1.5`(기본 2 아님) · round cap/join · **outline only(채움 없음)** · 단색 **웜잉크**(순검정·쿨그레이 금지, 크림/우드에 맞춰 웜톤) · 인장적색은 **화면당 1회 액센트만**(활성 스텝·전화 CTA) · 사이즈 **고정 스케일 20/24/32~40**(자유크기 금지) · 텍스트 라벨과 항상 페어링.
- **필요처**: 매장 정보(map-pin·clock·phone·car), 창업 절차 스텝, 본사지원(공급·교육·물류·표준화), 먹는법 스텝.
- **차별화 핵심**: 표준 세트에 없는 **브랜드 전용 line glyph 2~4개 자작**(보리밥 그릇·비빔 동작·장(醬)·낙관) — 동일 1.5px 스트로크로. "일관된 세트 + 소수 커스텀"이 안-템플릿 신호 1순위.
- **넘버링**: 절차·약속은 상페식 **01~04 (명조 큰 숫자 또는 인장적색 pill)**. 한자 넘버링은 메인에서 폐기 → 안 씀.
- 참고(절제 벤치마크): 정식당·Noma·Aesop/Le Labo = 아이콘 거의 안 쓰고 **괘선+여백**으로 구획. 매장 정보는 아이콘카드 그리드보다 **괘선 2열 목록(ruled directory)**이 프리미엄.

### 10F. 무료/라이선스 문양·에셋 소스 (리서치 확정)
> 규칙: 상업 웹엔 **공공누리 제1유형(상업+변경 모두 허용)**만. 출처표시는 전 유형 필수(푸터 한 줄).
- 문화포털 전통문양 `culture.go.kr/tradition/traditionalDesignPatternList.do` (대부분 1유형, 개별 확인)
- 공공누리 KOGL `kogl.or.kr` (유형별 상이, 1유형만 상업+변경)
- 국립중앙박물관 e뮤지엄 `emuseum.go.kr` (유물 고화질, 상당수 1유형)
- 국가유산 지식이음 `portal.nrich.go.kr` / 디자인DB `designdb.com`
- **가공 필수**: 원본은 풀컬러 단청(관광지 톤) → 다운로드 후 **단색화(우드/적색)+초저투명도(4~8%)**로 워터마크화해야 프리미엄 전환(1유형=변경 허용).

### 10G. 수채 일러스트 재활용 파이프라인 (브랜드 최대 무기)
"우리의 이야기" 수채(게·참기름·나물·보리·맞잡은 손)를 섹션 연결 조직으로:
1. 추출: pypdfium2 300dpi 렌더 → 일러스트 영역 크롭 (벡터 있으면 SVG가 최상)
2. 배경: **완전 투명화보다** 웹 배경을 같은 크림톤으로 두고 `mix-blend-mode: multiply` → 수채 여백이 배경에 자연스레 녹음(가장 손 덜 감). 필요시 higgs `remove_background`.
3. 배치: 스팟(100%, 카피 옆) / 배경 워터마크(opacity 8~15%, 코너 bleed, multiply) / 섹션 앵커(스크롤 따라 이야기 흐름)
4. **색 검수**: 배치 후 실제 화면 스샷 눈으로(화이트밸런스 튐 확인) — 헤드리스 통과 ≠ 시각검증
> 원칙(복순도가式): "전통 요소는 넣을 때마다 하나 빼라. 크림·우드·여백이 90%, 낙관·수채·명조가 10%로 격을 완성."

### 10D. "카드 그리드 남발" 탈피 (밋밋함 방지)
- menu 정식 = 균일 카드보다 **상페식 에디토리얼**(사진 + 화살표 라벨 + 명조 설명)도 혼용
- brand 재료 = 메인 **`.ing-strip` 오프셋** 레이아웃 재사용(높이 엇갈림)
- 신뢰 지표 = 딱딱한 배지 나열 대신 **아이콘+짧은 팩트 + 실물 사진(인증서·상패)** 조합

### 10E. 정직한 리스크 & 한계
- **일러스트 원본 아트는 내 약점**(메모리): 게·보리 수채화/선화는 **기존 자산 추출·재사용**으로 해결. 새 일러스트가 필요하면 디자이너/AI 생성 별도 트랙.
- 수채화 일러스트는 크림 배경에 얹혀 있어 **투명 추출 작업** 필요(PDF→PNG 후 배경 처리).
- **"괜찮게 나오는지"의 확답 = 렌더**. 디자인 요소를 실제로 얹은 히어로+섹션 1장을 시각 확인해야 최종 판정.
